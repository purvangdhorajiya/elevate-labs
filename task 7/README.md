# Task 7: Power BI Mini Dashboard

Dataset: Global Superstore Dataset

Dashboard Components:
- KPI Cards: Total Sales, Total Profit, Top Category
- Bar Chart: Sales by Category
- Bar Chart: Sales by Region
- Line Chart: Profit Trend Over Time
- Slicers: Region, Segment, Year

Insights:
1. Technology is the top revenue category.
2. West region leads in total sales.
3. Profit trend shows steady growth.

Outcome:
Learned to build interactive dashboards using Power BI with KPIs, slicers, and business visuals.
